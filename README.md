# 🐍 Curso de Python - Aplicativo Web

Bem-vindo ao repositório do meu primeiro aplicativo web focado no aprendizado de Python! Este projeto foi desenvolvido para demonstrar a aplicação prática da linguagem e está hospedado no Streamlit Cloud.

## 🚀 Acesse o Aplicativo
Você pode visualizar o projeto em execução aqui:
[Link do seu app no Streamlit](https://curso-python-eupuljyue3tp8d5vrrcrfm.streamlit.app)

## 🛠️ Tecnologias Utilizadas
* **Python**: Linguagem principal.
* **Streamlit**: Framework para criação da interface web.
* **GitHub**: Controle de versão e hospedagem do código.

## 📖 Sobre o Projeto
Este aplicativo é uma ferramenta interativa criada durante o meu aprendizado em Análise e Desenvolvimento de Sistemas. O objetivo é oferecer uma interface amigável para quem está começando a dar os primeiros passos no mundo da programação.
